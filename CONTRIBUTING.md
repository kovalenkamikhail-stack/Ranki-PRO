# Contributing

This repository is currently maintained by the owner and trusted local automation only.

## External Contributions

- External pull requests, unsolicited patches, and drive-by changes are not part of the default workflow.
- Public PRs from forks may be closed without review.
- Automated merge flow is reserved for trusted `codex/*` branches pushed by the repository owner workflow.

## Safety Expectations

- Do not submit secrets, credentials, private datasets, or personal data.
- Do not include local environment exports, browser storage dumps, or machine-specific files.
- If a change depends on private reference material, sanitize it before sharing.

## Preferred Path

- Use issues or direct owner communication first if coordination is needed.
- Owner-approved collaborator work should follow the same branch-first, check-gated flow as local automation.
